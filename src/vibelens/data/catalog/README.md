# VibeLens bundled catalog

This directory is regenerated by `scripts/build_catalog.py`. Do not hand-edit.

- `agent-*.json` — byte-for-byte copies of the `agent-tool-hub` output. Byte offsets in `catalog-offsets.json` depend on these bytes exactly. Any reformat breaks detail-view loading.
- `catalog-summary.json` — summary projection of all items, loaded into memory at startup.
- `catalog-offsets.json` — `{extension_id: [extension_type, byte_offset, byte_length]}` used to seek into `agent-*.json` for detail/install.
- `manifest.json` — metadata about the bundle (generation date, source dir, item counts).
